感謝香港網友( 我咁靚仔點解要受咁多苦 )，發現並開發控制器用Script。

感謝 Eternity Lam 為OHC-PC01提供TITAN TWO script檔 。
可透過TITAN TWO 在PS4上使用 三鶯重工 OHC-PC01/OHC-PC01A搖桿。

[使用方式]
-TITAN TWO部分（T2）
1.將T2的 PROG 接口連接至電腦。
22.先從TITAN TWO官網下載 Gtuner IV(https://www.consoletuner.com/titan-two-downloads/)。
3.解壓縮後執行Gtuner.exe主程式。
4a.開啟 DenShaDeGo_PS4_V1.xx.gpc 檔案。
4b.點選 File -> Install to -> Memoty Solt1 、Memoty Solt1 (Memoty Solt1 為你要安裝的儲存位置) 。
5a.開啟 DenShaDeGo_PS4_V1.xx(103).gpc 檔案。
5b.點選 File -> Install to -> Memoty Solt2 (Memoty Solt2 為你要安裝的儲存位置) 。
6.上傳完畢後移除電腦連線。
7.將T2 的OUTPUT連接至PS4前方USB，T2左邊USB1連接至OHC-PC01，右邊UBS2連接至PS4手把。

-PS4/PS5部分
1.在PS4/PS5設定中將手把連線方式設為 USB （非藍芽）。
2.啟動遊戲。
3.進入遊戲設定，將遊戲操作設定
  (1)操作タイプ → スタンダード
  (2)入力タイプ → ダイレクト
4.載入線路時，  
  (1) 將方向開關至中
  (2) 設定TITAN TWO數字旁按鈕，調整至1 或 2(103系用）

-操作方式
黑色按鈕：○ 喇叭 
黃色按鈕：△ 指差確認
白色按鈕：╳ 減光 ON/OFF
方向開關-前 暫停
方向開關-後 開啟雨刷


[日本語]
香港のネチズン（我咁靚仔點解要受咁多苦）がコントローラー用のスクリプトを発見して開発してくれたことに特に感謝します。

OHC-PC01用の「TITANTWO」スクリプトファイルを提供してくれたEternityLamに感謝します。
PS4からTITANTWOまで、三鶯重工OHC-PC01/OHC-PC01Aシリーズのジョイスティックを使用できます。

[使い方]
TITAN TWO（T2）の設定
1.T2の PROG ポートをコンピューターに接続します。
2. TITAN TWOの公式ウェブサイト（https://www.consoletuner.com/titan-two-downloads/）から「GtunerIV」をダウンロードします。
3. Gtuner.exeのプログラムを解凍して実行。
4a.DenShaDeGo_PS4_V1.xx.gpc ファイルを開きます。
4b.File -> Install to -> Memoty Solt1 をクリックします
5a.DenShaDeGo_PS4_V1.xx(103).gpc ファイルを開きます。
5b.File -> Install to -> Memoty Solt2 をクリックします
6.アップロード後、コンピューター接続を削除します。
7.T2の出力をPS4 / PS5の前面USBに接続し、T2の左側のUSB1をPS4/PS5のOHC-PC01に接続し、T2の右側のUBS2をPS4/PS5コントローラーに接続します。

[PS4/PS5での設定]
1. PS4/PS5の「設定」「周辺機器」「通信方法」から「USBケーブルを使う」を選択
2.ゲームを開始します。
3.ゲーム設定に移動し、ゲーム操作設定
   (1)タイプ → スタンダード
   (2)入力タイプ → ダイレクト.
4.ゲームが始まる前に、
   (1)コントローラー-方向スイッチ（右上部分）を中央（切）に回します
   (2)TITAN TWOアダプター番号パネルの横にあるボタンを押して1または2(103系用)に設定します

-操作方法
黒：○ 警笛を鳴らす
黄：△ 指差し確認操作
白：╳ 減光 ON/OFF
前：OPTIONS
後：□ ワイパー

[English]
Special thanks to Hong Kong netizens (我咁靚仔點解要受咁多苦)to discover and develop Script for controller.
Your hard work makes everyone enjoy the joy of the game.

Thanks for Eternity Lam for providing "TITAN TWO" script files for OHC-PC01.
You can use SANYING OHC-PC01/OHC-PC01A series joystick on PS4/PS5 through TITAN TWO.

[How to use]
Setting on TITAN TWO Adaptor (T2)
1. Connect the Prog port of T2 to the computer.
2. Download “Gtuner IV” from TITAN TWO official website (https://www.consoletuner.com/titan-two-downloads/).
3. Unzip and execute the program of Gtuner.exe
4a open DenShaDeGo_PS4_V1.xx.gpc file.
4b. Click file -> install to -> Memory Solt1.
5a open DenShaDeGo_PS4_V1.xx(103).gpc file.
5b. Click file -> install to -> Memory Solt2.
6. Remove the computer connection after uploading.
7. Connect the output of T2 to the front USB of PS4/PS5, USB1 on the left of T2 connect to OHC-PC01 of PS4/PS5, UBS2 on the right of T2 Connect to PS4/PS5 Controller.

[Setting on PS4 ]
1. Set the Controller connection method as USB in PS4/PS5 settings.
2. Start the game.
3. Go to game settings, set the game operation setting from 
   (1)タイプ → スタンダード
   (2)入力タイプ → ダイレクト.
4. Before the game started,
   (1) Controller - Turn the direction switch (Top right part) to the middle (切)   
   (2) Pressing the button on T2 to 1 or 2(103 kei use)

[How to play on Controller]
Black：○ 警笛を鳴らす
Yellow：△ 指差し確認操作
White：╳ 減光 ON/OFF
Direction switch FRONT：OPTIONS
Direction switch BACK：□ ワイパー
